## About Me

I'm a CS grad student at the **University of Illinois Chicago** (GPA 3.9) building production-grade AI systems. My work sits at the intersection of LLM infrastructure and distributed backend engineering. I care about systems that are fast, reliable, and actually ship.

Before grad school, I built backend infrastructure at a startup processing **50K+ daily transactions** with 99.9% uptime. Now focused on LLM agents, RAG systems, and scalable ML serving.
